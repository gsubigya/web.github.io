## 🎯 Purpose

Online scams disguised as games, lotteries, and betting platforms are one of the fastest-growing vectors of digital fraud. This project helps:

- **Educate** users on how deceptive game-based scam interfaces look and operate
- **Simulate** common scam mechanics in a safe, consequence-free environment
- **Train** individuals and security teams to identify red flags before falling victim
- **Raise awareness** within communities about the psychological tactics used by fraudulent platforms
